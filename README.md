# Godwin Sozo Kumah Portfolio

A modern developer portfolio for Godwin Sozo Kumah, also known as s3yram.

Built with:

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- lucide-react icons
- Vercel deployment

## Focus

This portfolio presents Godwin as a front-end and full-stack developer building practical systems for real businesses, including dashboards, payment flows, restaurant ordering systems, student portals, booking systems, and admin tools.

## Featured Projects

- Chiq-N-Grill Ordering System
- Dues Management System
- Royal Hawk Hotel Website
- HTH Hospital Map / Intern System
- AgroConnect
- Unlimited Edge Works

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Deployment

Deploy directly on Vercel by importing this GitHub repository.
